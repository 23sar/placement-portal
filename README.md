🎓 Campus Placement Portal

A simple and practical full-stack web application designed to make campus recruitment smoother for both students and companies. It helps companies post jobs and manage applicants, while students can easily explore opportunities and apply—all in one place.

🚀 What it does
🔐 Secure login and registration using JWT
🛡 Separate roles for Students and Companies
💼 Companies can post job openings
📄 Students can browse and apply for jobs
🌙 Clean UI with dark mode support
📱 Responsive design that works across devices


🛠 Tech Stack
Backend: Node.js, Express.js
Database: MySQL
Frontend: HTML, CSS, JavaScript
Authentication: JWT


⚙️ How to run locally
git clone https://github.com/23sar/placement-portal.git
cd placement-portal/server
npm install
node server.js


💡 Why I built this
This project was built to understand how real-world placement systems work and to get hands-on experience with backend development, authentication, and database design. It focuses on solving a practical problem while keeping the system scalable and easy to extend.
