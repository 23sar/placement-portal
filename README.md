# 🎓 Campus Placement Portal

A full-stack web application to streamline campus recruitment by connecting students and companies.

---

## 🚀 Features

- 🔐 JWT Authentication (Login/Register)
- 🛡 Role-based access (Student / Company)
- 💼 Job posting (Company)
- 📄 Job application (Student)
- 🌙 Dark Mode UI
- 🎨 Modern responsive UI

---

## 🛠 Tech Stack

- Backend: Node.js, Express.js  
- Database: MySQL  
- Frontend: HTML, CSS, JavaScript  
- Auth: JWT  

---

## ⚙️ Setup Instructions

```bash
git clone https://github.com/23sar/placement-portal.git
